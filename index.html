FlashUSDT Simulation & Token Behavior Analysis

🔍 Overview

This project aims to simulate deceptive token behaviors using a custom ERC-20 token "FlashUSDT". The system includes real and simulated transfers, cross-chain testnet deployment, and notification logic to analyze visibility versus validity.

⚙️ Deliverables

  • Real ERC-20 token contract on Sepolia, BSC Testnet, Polygon
  • Simulated "flash" transfer behavior (event-only)
  • Telegram bot for instant alerts
  • Frontend UI (optional)
  • Documentation, flow diagrams, walkthrough
